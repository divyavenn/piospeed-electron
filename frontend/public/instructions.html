PioSpeed Instructions

Getting Started

Welcome to PioSpeed, an Electron-based interface for PioSOLVER. This application allows you to interact with PioSOLVER through a modern graphical interface.

Note: This application requires Python 3 to be installed on your system. The application will check for Python and notify you if it's not found.

Setting Up

  1. Select your PioSOLVER executable by clicking the "Select Solver" button in the settings panel.
  2. Once the solver is selected, the application will automatically initialize the connection.
  3. You can also configure mapping files and other settings in the settings panel.

Using Commands

The application provides access to various commands available in PioSOLVER:

  1. Select a command from the dropdown menu.
  2. Fill in the required parameters for the selected command.
  3. Click "Execute" to run the command.
  4. View the results in the output panel.

Input and Output

The application may request input from you during command execution:

  • Text input: Type your response in the text field and press Enter.
  • File selection: Use the file browser to select the required file.
  • Directory selection: Use the directory browser to select the required folder.

Output from commands will be displayed in the output panel, including:

  • Command results
  • Status updates
  • Error messages

Troubleshooting

Python Issues

If the application reports issues with Python:

  • Make sure Python 3 is installed and in your system PATH.
  • Try restarting the application.
  • Check the console log for more detailed error messages.

PioSOLVER Connection Issues

If the application cannot connect to PioSOLVER:

  • Verify the path to the PioSOLVER executable is correct.
  • Ensure PioSOLVER is not already running in another process.
  • Check if your firewall is blocking the connection.
Warning: Always make sure to close the application properly to ensure the PioSOLVER process is terminated correctly.
